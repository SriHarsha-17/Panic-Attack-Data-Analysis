# 📊 Panic Attack Data Analysis

## 📌 Overview

This project analyzes panic attack data to identify patterns in symptoms, triggers, and lifestyle factors influencing mental health outcomes. The analysis leverages **Snowflake (SQL)** for data processing and **Power BI** for building interactive dashboards.

The objective is to derive actionable insights that support better understanding of panic attack behavior and contributing factors.

## 🛠️ Tech Stack

* **Snowflake (SQL)** – Data querying and analysis
* **Power BI** – Dashboard development and visualization
* **Power Query Editor** – Data transformation and preprocessing

---

## 📂 Dataset Description

The dataset consists of multiple dimensions related to panic attacks, including:

* **Symptoms:** Dizziness, Sweating, Trembling, Chest Pain, Shortness of Breath
* **Demographics:** Age Group, Gender
* **Lifestyle Factors:** Smoking, Alcohol Consumption, Exercise Frequency
* **Medical History:** Anxiety, Depression
* **Severity Levels:** Panic Score (Low, Medium, High)

---

## 📊 Key Insights

### 🔹 Symptom Distribution

* **Sweating (836 cases)** and **Shortness of Breath (746 cases)** are the most frequently reported symptoms
* High occurrence of **Dizziness and Trembling** indicates common physiological responses
* **Chest Pain** is present in a significant number of cases

(Insights derived from dashboard visuals) 

---

### 🔹 Behavioral & Lifestyle Patterns

* Variations in **sleep duration** and **alcohol consumption** show noticeable patterns across patients
* Lifestyle indicators such as **smoking and exercise frequency** provide additional context for analysis

---

### 🔹 Trend Analysis

* Panic attack duration varies widely across individuals
* Sleep patterns exhibit clustered distributions across different hour ranges
* Alcohol consumption trends fluctuate across patient groups

(Trend observations from dashboard analysis)

## 📈 Dashboard Highlights

* Multi-page interactive dashboard (3 report views)

* Comparative analysis of symptoms across different categories
* Trend visualization using line and bar charts
